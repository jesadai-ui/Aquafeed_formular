1. กำหนดสเปคอาหาร (มาตรฐานขั้นต่ำ/ขั้นสูง)

2. ตารางวัตถุดิบและสัดส่วนที่ต้องการ

📚 อ้างอิงสูตรการคำนวณระดับสากล:
• % คาร์โบไฮเดรต (NFE) = 100 - (%ความชื้น + %โปรตีน + %ไขมัน + %เยื่อใย + %เถ้า)
• ค่าพลังงานย่อยได้ (Digestible Energy: DE) อ้างอิงจากหลักการประเมินค่าพลังงานสำหรับสัตว์น้ำ (NRC, 1993 / 2011) โดยใช้ค่าสัมประสิทธิ์พลังงาน:
  DE (kcal/100g) = (%โปรตีน × 4.0) + (%ไขมัน × 9.0) + (%NFE × 3.0)
กลุ่ม / ชนิดวัตถุดิบ % ความชื้น % โปรตีน % ไขมัน % เยื่อใย % เถ้า % คาร์บ (NFE) DE (kcal/100g) ราคา (บาท/กก.) สัดส่วนที่ใช้ (%) จัดการ

4.3 บทวิเคราะห์เศรษฐศาสตร์และต้นทุนการเลี้ยง

  • ต้นทุนค่าวัตถุดิบอาหาร: - บาท/กิโลกรัม
  • ต้นทุนค่าอาหารต่อการได้เนื้อ 1 กก. (ตาม FCR): - บาท
  • ต้นทุนการผลิตสัตว์น้ำรวมทั้งหมด (เทียบเป็น 100%): - บาท/กก.
  • ราคาขายสัตว์น้ำที่คาดการณ์: - บาท/กก.
  • บทสรุปผลการเลี้ยง (กำไร / ขาดทุน):
    - บาท/กก.
